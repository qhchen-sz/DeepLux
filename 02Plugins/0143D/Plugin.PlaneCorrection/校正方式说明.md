# 平面校正方式说明

本文档说明 `Plugin.PlaneCorrection` 插件中的三种校正方式：快速校正、投影校正、点到面距离图。

## 共同输入

插件执行时主要使用这些数据：

- 当前输入高度图：默认取第 1 通道作为高度 Z 图。
- 基准平面：可以来自“引用基准平面图”，也可以来自手动输入的 `Nx/Ny/Nz/D` 平面方程。
- ROI 区域：用于限定当前图上的有效区域。
- 平移 Z：可选的固定高度偏移，只改变输出高度，不改变平面角度。

基准平面的作用是提供“目标姿态”。当前输入图可能是倾斜的，插件会根据不同模式，把当前图与基准平面建立关系。

ROI 的意义需要分模式理解：

- 快速校正：ROI 用于拟合当前工件平面。
- 投影校正：ROI 用于拟合当前工件平面，也作为 3D 旋转中心参考。
- 点到面距离图：ROI 主要用于统计/检测区域，不需要用 ROI 拟合当前平面后再计算距离。

如果不启用 ROI，插件使用当前图像的整个有效区域。

## 1. 快速校正

### 含义

快速校正是在高度图上做 Z 方向趋势补偿。它把当前图中的整体倾斜趋势去掉，再叠加到基准平面姿态上。

这种方式速度快，适合倾斜不大、只需要把高度图趋势拉平或对齐基准平面的场景。

### 计算过程

1. 在当前输入高度图上，根据 ROI 拟合当前平面。

   HALCON 一阶面形式：

   ```text
   Z当前平面 = Alpha * (row - row0) + Beta * (col - col0) + Gamma
   ```

2. 获取基准平面。

   如果使用引用基准平面图，插件会对基准平面图做整体平面拟合。

   如果使用手动参数，插件会把 `Nx/Ny/Nz/D` 转成高度图上的一阶平面。

3. 计算当前高度图相对当前拟合平面的残差。

   ```text
   残差图 = 原始高度图 - 当前拟合平面图
   ```

   残差图表示局部凸起、凹陷、缺陷、真实形貌细节。

4. 计算当前平面中心相对基准平面的高度差。

   ```text
   高度差 = 当前平面在 ROI 中心的高度 - 基准平面在 ROI 中心的高度
   ```

5. 生成校正后的高度图。

   ```text
   输出图 = 残差图 + 基准平面图 + 高度差 + 平移Z
   ```

### 特点

- 优点：速度快，逻辑稳定，对普通高度趋势补偿很实用。
- 优点：会保留当前工件相对基准平面的高度差，不会强行把中心高度抹平。
- 限制：它没有改变真实 X/Y 几何位置，只是在高度图 Z 值上做补偿。
- 限制：如果倾斜角较大，例如 30 度左右，结果不等价于真实 3D 姿态校正。

### 适用场景

- 工件倾斜较小。
- 只关心高度趋势补偿。
- 后续算法主要看相对高度、平面度、局部缺陷。
- 对真实 3D 几何投影精度要求不高。

## 2. 投影校正

### 含义

投影校正是真实 3D 姿态校正。它会把高度图按 X/Y/Z 分辨率转换成物理 XYZ 点云，然后把当前平面的法向量旋转到基准平面的法向量，最后再投影回高度图。

这种方式更适合工业 3D 视觉中倾斜角较大、需要严格保留空间几何关系的场景。

### 为什么需要 X/Y/Z 分辨率

投影校正必须知道像素坐标和高度值对应的真实物理比例：

```text
X = col * X分辨率
Y = row * Y分辨率
Z = zPixel * Z分辨率
```

如果 X/Y/Z 分辨率不是真实比例，旋转角度和几何关系都会不准确。

快速校正和点到面距离图一般不需要用户关心 X/Y/Z 分辨率，所以界面中只有选择“投影校正”时才显示分辨率设置。

### 计算过程

1. 根据 ROI 在当前输入高度图上拟合当前平面。

2. 获取基准平面。

   基准平面可以来自引用基准平面图的拟合结果，也可以来自手动输入的标准平面方程。

3. 把当前平面和基准平面都转换成标准物理坐标下的平面法向量。

   标准平面形式：

   ```text
   Nx * X + Ny * Y + Nz * Z = D
   ```

4. 为整张高度图生成物理坐标图。

   ```text
   X图 = col * X分辨率
   Y图 = row * Y分辨率
   Z图 = 原始高度图 * Z分辨率
   ```

5. 使用 `XyzToObjectModel3d` 把 X/Y/Z 图转换为 3D 点云。

6. 构造 3D 旋转矩阵。

   旋转目标是：

   ```text
   当前平面法向量 -> 基准平面法向量
   ```

   当前实现围绕 ROI 中心对应的 3D 点旋转，所以会尽量保留当前工件相对基准面的中心高度差。

7. 使用 `AffineTransObjectModel3d` 对整张点云做 3D 变换。

8. 使用 `ObjectModel3dToXyz(..., "from_xyz_map", ...)` 把变换后的点云写回原高度图网格。

9. 取变换后的 Z 图，并除回 `Z分辨率`，使输出高度单位与输入高度图保持一致。

10. 如果启用了平移 Z，再在输出图上加固定 Z 偏移。

### 特点

- 优点：是真实 3D 几何姿态校正。
- 优点：适合较大倾斜角，不只是简单的 Z 值趋势补偿。
- 优点：姿态被校正，同时尽量保留当前工件相对基准面的高度差。
- 限制：必须填写正确的 X/Y/Z 物理分辨率。
- 限制：依赖 HALCON 的 3D 点云转换和 `from_xyz_map` 回写能力。
- 限制：计算量比快速校正更大。

### 适用场景

- 工件或基准面倾斜明显。
- 需要更严格的 3D 几何校正。
- 高度图的 X/Y/Z 物理比例明确。
- 后续测量依赖真实空间姿态，而不只是局部高度趋势。

## 3. 点到面距离图

### 含义

点到面距离图不是“姿态校正后的高度图”，而是计算当前高度图中每个点到基准平面的距离。

它输出的是一张距离图，图中每个像素表示该点相对基准平面的法向距离。

这种方式更适合直接做平面度、凸起、凹陷、缺陷检测。

### 计算过程

1. 获取基准平面。

   基准平面可以来自引用基准平面图，也可以来自手动输入的 `Nx/Ny/Nz/D`。

2. 根据基准平面生成与当前输入图同尺寸的基准平面图。

   ```text
   Z基准平面(row, col)
   ```

3. 当前输入高度图减去基准平面图。

   ```text
   Z差值图 = 原始高度图 - 基准平面图 + 平移Z
   ```

4. 把 Z 方向差值转换为近似法向距离。

   当前实现使用基准平面法向量的 `Nz` 分量：

   ```text
   距离图 = Z差值图 * Nz
   ```

5. 输出距离图。

### ROI 在点到面距离图中的作用

点到面距离图计算每个点到基准平面的距离时，并不需要先拟合当前平面。

但是 ROI 仍然有意义：

- 限定哪些点参与最大偏差、最小偏差、平面度、RMS 统计。
- 限定显示出来的检测区域。
- 避开边缘、背景、夹具、无效点或不想统计的区域。

因此在点到面距离图模式下，ROI 更准确的理解是“测量区域”或“统计区域”，不是“当前平面拟合区域”。

### 特点

- 优点：直接表达每个点相对基准平面的距离。
- 优点：适合平面度、凸起/凹陷、缺陷检测。
- 优点：不需要拟合当前平面后再做姿态校正。
- 限制：输出不是普通意义上的校正后高度图。
- 限制：如果后续插件期望输入仍是姿态校正后的 Z 高度图，需要确认是否适合使用该模式。

### 适用场景

- 目标是判断某个区域相对基准面高了多少或低了多少。
- 目标是计算平面度、最大凸起、最大凹陷、RMS。
- 目标是做缺陷检测，而不是生成一张姿态校正后的高度图。

## 三种方式对比

| 方式 | 输出是什么 | 是否拟合当前平面 | 是否真实 3D 投影 | 是否需要真实 X/Y/Z 分辨率 | 典型用途 |
| --- | --- | --- | --- | --- | --- |
| 快速校正 | 校正后高度图 | 是 | 否 | 否 | 快速高度趋势补偿 |
| 投影校正 | 校正后高度图 | 是 | 是 | 是 | 大倾角、真实 3D 姿态校正 |
| 点到面距离图 | 到基准面的距离图 | 否，当前代码共用流程中会拟合但输出不依赖当前平面 | 否 | 通常不需要 | 平面度、凸起/凹陷、缺陷检测 |

## 选择建议

- 默认优先使用“快速校正”：速度快，适合常规高度图补偿。
- 倾斜角较大、并且知道真实 X/Y/Z 分辨率时，使用“投影校正”。
- 目标是判断每个点离基准平面多远时，使用“点到面距离图”。

## 显示设置说明

- 显示校正图：只影响界面显示。勾选后会把输出结果做 0 到 255 的可视化缩放并显示，不改变 `CorrectedImage` 的真实高度数据。
- 显示区域：显示参与统计/检测的区域。启用 ROI 时显示 ROI 与图像有效区域的交集；不启用 ROI 时通常显示整图有效区域。
- 显示结果文字：在显示窗口中叠加平面度和 RMS 文本。
